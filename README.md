# Практическое задание 1 курса "Основы разработки игр на Unity" проекта "Открытое образование"

**Концепция игры**: Создание 2D платформера, где персонаж прыгает по перекладинам уже по вертикали, а не по горизонтали. Используйте все изученные возможности, применяя любые доступные ассеты.

1. Создайте проект «Project1» в Unity.
2. Обязательно присутствие заднего фона, персонажа и перекладины
3. Количество и расположение элементов может быть произвольным.

**Важно**: в данном макете можно использовать произвольный текст и изображение.
